# avp_wiki 
avp Autonomous Valet Parking 自动代客泊车  
自动/自主泊车相对容易在量产车中落地实现，同时也是刚需，是一个比较有意思的事情(感知+定位+规控)，作为入门选手这里总结下一些收集的资料
## 1. 缩写及术语

| :----: | :----: | :----: |
